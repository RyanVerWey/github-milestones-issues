# Change Log

All notable changes to the "github-milestones-issues" extension will be documented in this file.

## [0.1.0] - 2025-11-23

### Added
- Edit issue title and description
- Edit milestone title and description
- Assign issues to milestones (or remove assignment)
- Close/reopen issues directly from context menu
- Close/reopen milestones directly from context menu
- Context menu on all items with edit, state management, and open actions
- Quick pick selector for milestone assignment with visual icons

### Changed
- Improved user interface with inline edit buttons
- Enhanced context menus for better workflow

## [0.0.1] - 2025-11-23

### Added
- Initial release
- GitHub authentication
- View milestones and issues in sidebar
- Create new milestones and issues
- Open milestones and issues in browser
- Automatic repository detection
