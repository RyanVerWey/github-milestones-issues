### How to Install/Update the VSIX

#### Option 1: Via Command Line (Recommended)
```bash
code --install-extension /Users/ryanverwey/GitHubProjects/github-milestones-issues/github-milestones-issues-0.1.0.vsix
```

#### Option 2: Via VS Code UI
1. Open VS Code
2. Go to Extensions view (Cmd+Shift+X)
3. Click the "..." menu at the top of the Extensions view
4. Select "Install from VSIX..."
5. Navigate to and select: `github-milestones-issues-0.1.0.vsix`
6. Reload VS Code when prompted

### New Features to Test

#### Edit Issues
1. Right-click any issue in the sidebar
2. Select "Edit Issue"
3. Modify title and description
4. Changes sync immediately

#### Assign to Milestone
1. Right-click any issue
2. Select "Assign to Milestone"
3. Choose from list of open milestones
4. Select "Remove from milestone" to unassign

#### Close/Reopen Items
- Right-click any issue or milestone
- Select "Close Issue/Milestone" or "Reopen Issue/Milestone"
- State updates instantly

#### Edit Milestones
1. Right-click any milestone
2. Select "Edit Milestone"
3. Update title and description
